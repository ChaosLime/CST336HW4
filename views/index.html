What is Free and Open Source Software?

<% include partials/nav %> <% include partials/faker %>

Just what is it?

Free and Open source Software is software with source code that anyone can inspect, modify, and enhance.

"Source code" is the part of software that most computer users don't ever see; it's the code computer programmers can manipulate to change how a piece of software—a "program" or "application"—works. Programmers who have access to a computer program's source code can improve that program by adding features to it or fixing parts that don't always work correctly.

What makes it different?

Some software has source code that only the person, team, or organization who created it—and maintains exclusive control over it—can modify. People call this kind of software "proprietary" or "closed source" software.

The intent is for its original authors are the only ones that that can legally copy, inspect, and alter their software. For a person to use any proprietary software, computer users have to agree to software agreements or licenses and will limit the user to the intended agreement be it that they can only use it, but not change, modify or redistribute it, and in some cases limit the user to a time limit.

Free and Open Source Software is different. The authors allow source code to be available anyone that would like to view that code, copy it, learn from it, alter it, or share it. Much like that of closed source software, there are license agreemnts, but theses agreemnts differ dramatically from those of propriery licenses. See Licensing for more details on the most common Open source licenses.
